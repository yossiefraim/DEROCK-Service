/albums

Add Songs -

Adding song to user favorites by song id and user id


Path:

/addSongs


Method:

POST


Parameters:

id (user id), addSongId (song id)


Active Function:

addSong(user_id,add_song_id)


Remove Songs -

Removing song from user favorites by song id and user id


Path:

/addSongs


Method:

POST


Parameters:

id (user id), removeSongId (song id)


Active Function:

removeSong(user_id,remove_song_id)
